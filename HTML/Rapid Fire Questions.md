# HTML — Rapid Fire Questions

1. **Full form of HTML?** HyperText Markup Language.
2. **Which tag defines the document type?** `<!DOCTYPE html>`.
3. **Which tag holds page metadata?** `<head>`.
4. **Block or inline: `<div>`?** Block.
5. **Block or inline: `<span>`?** Inline.
6. **Which attribute provides alternate text for images?** `alt`.
7. **Which HTML5 tag is used for navigation links?** `<nav>`.
8. **Which tag embeds a video natively?** `<video>`.
9. **Attribute to open link in a new tab?** `target="_blank"`.
10. **Which input type restricts to numbers?** `type="number"`.
11. **Which tag defines a table row?** `<tr>`.
12. **Which tag defines a table header cell?** `<th>`.
13. **Which storage persists after browser/tab closes?** localStorage.
14. **Which storage clears when tab closes?** sessionStorage.
15. **Void element example?** `<br>`, `<img>`, `<hr>`, `<input>`.
16. **Attribute used to link CSS file?** `<link rel="stylesheet" href="...">`.
17. **Tag for the largest heading?** `<h1>`.
18. **Which meta tag ensures mobile responsiveness?** viewport meta tag.
19. **Attribute for custom data on elements?** `data-*`.
20. **Tag for line break?** `<br>`.
21. **Which tag groups form controls with a caption?** `<fieldset>` + `<legend>`.
22. **Global attribute for element ID?** `id`.
23. **Which tag defines an unordered list?** `<ul>`.
24. **Which tag defines an ordered list?** `<ol>`.
25. **Which attribute makes an image load lazily?** `loading="lazy"`.
26. **Script attribute that preserves execution order but doesn't block parsing?** `defer`.
27. **Script attribute that executes as soon as downloaded, order not guaranteed?** `async`.
28. **Which element embeds another HTML page?** `<iframe>`.
29. **Which tag represents a self-contained composition (e.g. blog post)?** `<article>`.
30. **Which HTML5 element gives native modal support?** `<dialog>`.
31. **Which attribute disables interaction/focus on a subtree?** `inert`.
32. **What does ARIA stand for?** Accessible Rich Internet Applications.
33. **Which tag is used for a dropdown list?** `<select>`.
34. **Character entity for `<`?** `&lt;`.
35. **Which HTTP method is idempotent, GET or POST?** GET.

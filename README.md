# Zero to Hero — SQL, HTML, CSS, JavaScript, Python

A complete interview-prep and reference collection covering 5 languages, each broken into 9 focused markdown files — from basic theory to managerial/HR rounds.

## 📁 Folder Structure

```
/SQL
/HTML
/CSS
/JavaScript
/Python
```

Each folder contains the same 9-file structure:

| # | File | What's Inside |
|---|---|---|
| 01 | `Zero_to_Hero_Theory.md` | Complete theory — basics to advanced concepts, with code examples |
| 02 | `Troubleshooting_Guide.md` | Common errors, causes, and fixes (Symptom → Cause → Fix format) |
| 03 | `RealTime_Production_Scenarios.md` | Real-world production incidents — Situation → Investigation → Root Cause → Fix → Prevention |
| 04 | `Interview_Questions_1to4_Years.md` | Interview questions targeted at 1–4 years of experience, with answers |
| 05 | `Latest_Market_Questions_2026_2027.md` | Modern/current interview trends and questions (2026–2027 market standard) |
| 06 | `Rapid_Fire_Questions.md` | Quick one-line Q&A for last-minute revision |
| 07 | `Company_Wise_Interview_Patterns.md` | Interview patterns by company category (service-based, product-based, fintech, startups) |
| 08 | `Managerial_Round_Questions.md` | Behavioral/managerial round questions (STAR-format guidance) |
| 09 | `HR_Round_Questions.md` | General HR round questions and tips |

## 📚 How to Use This

- **Learning from scratch?** Start with file `01` (theory) in each folder, work through examples.
- **Debugging a live issue?** Jump to file `02` (troubleshooting) or `03` (real-time scenarios) for a similar pattern.
- **Prepping for interviews?** Go in order: `04` → `05` → `06` → `07` → `08` → `09`, mapping to the actual interview funnel (technical rounds → managerial → HR).
- **Last-minute revision?** File `06` (Rapid Fire) in each folder is built for quick scanning.

## 🗂️ Languages Covered
1. **SQL** — relational databases, query writing, optimization, transactions
2. **HTML** — semantic markup, accessibility, forms, modern HTML5 APIs
3. **CSS** — layout (Flexbox/Grid), responsive design, modern CSS features
4. **JavaScript** — core language, async/event loop, modern ES6+ and 2026 patterns
5. **Python** — core language, OOP, concurrency (GIL/asyncio/multiprocessing), modern tooling

## ⚠️ A Note on "Company-Wise" and "Latest Market" Files
The company-wise interview pattern files describe **general, commonly reported patterns** by company category (service-based, product-based, fintech, startups) — not confirmed/leaked question banks from specific companies. The "Latest Market" files reflect the technology and interview trends most relevant heading into 2026–2027, based on how the industry has been evolving.

## ✅ Suggested Study Path (Full Zero-to-Hero)
1. Read the Theory file front to back for each language, coding along with every example.
2. Solve the practical/coding questions in the Interview Questions file yourself before checking answers.
3. Review the Troubleshooting + Real-Time Scenarios files — these are what separates "knows the syntax" from "can work in production."
4. Do a Rapid Fire pass a day or two before interviews.
5. Read Company-Wise Patterns to calibrate expectations for the specific type of company you're interviewing with.
6. Prepare 2–3 personal stories using the Managerial Round file (STAR format) — reuse them across HR round too.
7. Review HR Round questions and prepare your own closing questions for the interviewer.

Good luck! 🚀

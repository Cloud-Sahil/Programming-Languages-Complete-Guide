# CSS — Rapid Fire Questions

1. **Full form of CSS?** Cascading Style Sheets.
2. **Which property changes text color?** `color`.
3. **Which property changes background color?** `background-color`.
4. **Default `box-sizing` value?** `content-box`.
5. **Which display value hides an element and removes its space?** `display: none`.
6. **Which property hides an element but keeps its space?** `visibility: hidden`.
7. **Which positioning stays fixed relative to viewport?** `position: fixed`.
8. **Which positioning is relative to the nearest positioned ancestor?** `position: absolute`.
9. **Property for spacing between flex/grid items?** `gap`.
10. **Which axis does `justify-content` control in a row flex container?** Main axis (horizontal).
11. **Which axis does `align-items` control in a row flex container?** Cross axis (vertical).
12. **Unit relative to root font size?** `rem`.
13. **Unit relative to parent font size?** `em`.
14. **Selector with highest specificity: ID or class?** ID.
15. **Property to create rounded corners?** `border-radius`.
16. **Property for element shadow?** `box-shadow`.
17. **Pseudo-class for the first child element?** `:first-child`.
18. **Pseudo-element to insert content before an element?** `::before`.
19. **Property to control overflow of content?** `overflow`.
20. **Which CSS function is used for responsive typography combining units?** `clamp()`.
21. **Property to define grid columns?** `grid-template-columns`.
22. **Property to control flex item growth?** `flex-grow`.
23. **What does `z-index` control?** Stack order of overlapping elements.
24. **Which selector selects a parent based on its child? (modern)** `:has()`.
25. **Media feature that detects a user's reduced motion preference?** `prefers-reduced-motion`.
26. **Function to combine two colors natively?** `color-mix()`.
27. **CSS at-rule for responsive design based on screen size?** `@media`.
28. **CSS at-rule for defining cascade order?** `@layer`.
29. **Property to define custom variables?** Custom properties, e.g. `--main-color: blue;` accessed via `var(--main-color)`.
30. **What causes margin collapsing?** Adjacent vertical margins between block-level elements.
31. **Which property prevents text wrapping?** `white-space: nowrap`.
32. **Which property adds an ellipsis for truncated text?** `text-overflow: ellipsis`.
33. **Default flex-direction value?** `row`.
34. **What does `aspect-ratio` do?** Sets a fixed width-to-height ratio for an element.
35. **Which viewport unit adjusts dynamically for mobile browser UI?** `dvh`.
